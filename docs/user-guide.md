# User Guide

## 1. Duplicate Cleanup

1. Open the extension popup
2. Configure rules in Duplicate Cleanup:
   - Ignore query params
   - Ignore hash
   - Ignore protocol
   - Ignore www
3. Click Clean duplicates
4. The system keeps the earliest tab and closes other duplicates

## 2. Smart Tab Grouping

1. View automatic grouping results in Domain Groups
2. Click Expand/Collapse to inspect tabs inside each group
3. Each group shows:
   - Tab count
   - Estimated memory (MB)

## 3. Search

1. Enter keywords in Search
2. Select a search mode:
   - Normal: substring match
   - Fuzzy: character sequence match
   - Regex: expression match
3. Review highlighted results and locate the target tab

## 4. Workspace Management

1. Enter a name and click Save current
2. Select records from the list and click Restore selected
3. Click Export to save JSON
4. Click Import and choose a JSON file to restore
